# PulsePace-AI-Emergency-Assistance-System

**Overview:**

PulsePace is a voice-activated emergency assistance system designed to guide bystanders during critical medical situations. When a person collapses or stops breathing, panic often prevents people from performing proper CPR. PulsePace provides real-time voice guidance for CPR, while simultaneously sending the user's location and emergency details to nearby hospitals and ambulance drivers.
The system helps save lives by enabling quick response, guided CPR instructions, and faster emergency coordination.

**Problem Statement:**

During sudden medical emergencies such as cardiac arrest:
-> Bystanders panic and forget CPR steps.
-> Ambulances are not alerted immediately.
-> Lack of guidance delays life-saving action.
-> This delay can significantly reduce the chances of survival.

**Solution:**

PulsePace acts as a voice-based emergency assistant that:
-> Listens to the user through a voice input button.
-> Detects emergency phrases like “He is not breathing.”
-> Provides step-by-step CPR instructions in voice format.
-> Sends GPS location and user details to nearby hospitals and ambulance drivers.
-> Displays ambulance driver details on the screen for quick response.

**System Architecture:**

-> User presses the Voice Assistant button (+).
-> The system records the voice input.
-> Speech is converted to text using Speech Recognition.
-> Backend processes the emergency request.
-> CPR instructions are generated and converted to audio output.
-> User GPS location is sent to nearby hospitals and ambulance drivers.
-> Ambulance details are displayed on the interface.

**Conclusion:**

PulsePace is designed to assist people during critical medical emergencies by providing instant voice-guided CPR instructions and real-time emergency coordination. By combining voice technology, geolocation, and rapid communication with hospitals and ambulances, the system aims to reduce response time and increase the chances of saving lives.
This project demonstrates how technology can be used to create accessible, life-saving solutions that empower bystanders to take immediate action during emergencies.
